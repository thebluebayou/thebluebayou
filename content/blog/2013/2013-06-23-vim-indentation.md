Ctrl-P toggles paste mode when in Insert mode do not include the two /  
{% highlight bash html %}
set pastetoggle=/<C-P/> "Ctrl-P toggles paste mode when in Insert"
{% endhighlight %}

{% highlight bash html %}
:set paste
{% endhighlight %}

{% highlight bash html %}
:set nopaste
{% endhighlight %}
